Tag HTML พื้นฐาน
Tag ความหมาย รูปแบบการนำไปใช้งาน
< html> เริ่มต้นใช้งาน HTML < html> ...... < /html>
< title> ในส่วนตัวอักษรที่อยู่ในคำสั่งนี้จะอยู่ใน title bar ของ web page < title> ...... < /title>
< head> เป็นส่วนที่ใช้ให้รายละเอียดเกี่ยวกับ เว็บเพจหน้านี้ < head> ...... < /head>
< body แสดงเนื้อเรื่องของ web < body> ...... < /body>
< p> คำสั่งที่ใช้ในการจัดย่อหน้า < p> ...... < /p>
< br> คำสั่งที่ใช้ในการขึ้นบรรทัดใหม่ < br> ...... < /br>
< h> คำสั่งที่ใช้กับข้อความที่เป็นหัวเรื่อง < h> ...... < /h>
< hr> ขีดเส้นใต้ < hr size=...px noshade="">
< pre> ดลอกเนื้อหาจากที่อื่นมาทำให้เราไม่ต้องขึ้นบรรทัดใหม่ด้วยคำสั่ง br < pre> ..... < /pre>
< align> เลขโรมัน < align="...">
< b>.< strong> ตัวอักษรแบบตัวหนา < b> ...... < /b>
< i>,< em> ตัวอักษรแบบตัวเอียง < i> ...... < /i>
< small> ลดขนาดตัวอักษร < small> ...... < /small>
< mark> ไฮไลท์ส่วนที่สำคัญ < mark> ...... < /mark>
< mark style=color:"..."> ไฮไลท์ส่วนที่สำคัญและเปลี่ยนสีตัวอักษร < mark style=color:"...">
< u> ตัวอักษรแบบขีดเส้นใต้ < u> ...... < /u>
< sup> ตัวอักษรแบบตัวยกกำลัง < sup> ...... < /sup>
< ul> ตัวอย่าง < ul> ...... < /ul>
< li> ตัวอย่างย่อย < li> ...... < /li>
< ol> เลขโรมัน < ol type="i"> ...... < /ol>
< ol> เรื่มที่เลขนั้น < ol type="5"> ...... < /ol>
< img> แทรกรูปภาพ < img src="....." alt='...' width='...' height='...'>
< img src="URL"> แทรกลิงค์ < img src="................">
< a href> สร้าง hyperlink < a href=.........>ชื่อที่ต้องการใส่
< table> สร้างตาราง < table>...< /table>
< tr> ก่อนขึ้นหัวข้อในตาราง < tr> ... < /tr>
< th> หัวข้อในตาราง < th> ... < /th>
< td> ข้อมูลในตาราง < tr> ... < /tr>
< bg color> เปลี่ยนสี bg style="background-color:...;">
< rowspan> สร้างช่องในหลายแถว < rowspan="..."> ...... < /ol>
< colspan> สร้างช่องในหลายคอลัมน์ < ol type="i"> ...... < /ol>
< form> สร้างที่กรอกข้อมูล < form> ...... < /form>
< label> กรอกเป็นตัวอักษร < label> ...... < /label>
< input> ช่องใส่ข้อมูล < input> ...... < /input>
